## Adding New Screenshots

When adding screenshots:

1. Choose appropriate directory (ui-reference/ or design-mockups/)
2. Use descriptive filename: `feature-name-view-date.png`
3. Add entry to this catalog with:
   - File name
   - Date captured/created
   - Purpose/context
   - Related documentation links
